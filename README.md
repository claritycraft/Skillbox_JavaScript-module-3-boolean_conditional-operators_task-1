### Ru:
#### Задача 1
Напишите код для банкомата, который:

- хранит состояние, есть ли карта в банкомате, а также доступную сумму;
- принимает на вход сумму банковской операции;
- проверяет два условия: карта есть в банкомате и введённая сумма меньше или равна балансу на карте;
- если все условия выполняются, то выводит в консоль сообщение об успешной операции: «Операция выполняется»;
- если условия не выполняются, то выводит сообщение с ошибкой: «Операция отклонена».

#### Пример работы программы

В программе записано, что карта находится в банкомате и доступная сумма составляет 500 рублей.

Ввод: 1000
Вывод: Операция отклонена

### En:
#### Task 1
Write code for an ATM that:

- Stores the state of whether a card is inserted and the available balance
- Accepts the amount of the banking operation as input
- Checks two conditions:
- A card is present in the ATM
- The entered amount is less than or equal to the card balance
- If both conditions are met, it outputs a success message to the console:
"Transaction in progress"
- If the conditions are not met, it outputs an error message:
"Transaction declined"

#### Example program output
The program is set up with a card inserted and an available balance of 500 rubles.
Input: 1000
Output: Transaction declined
